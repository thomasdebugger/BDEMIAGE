La LAN 2017 de l'IM²AG arrive!

Le 16 mars 2017

Pour la 8ème année, le BDE MIAGE est fier de vous proposer la LAN de l'UFR IM²AG de Grenoble.
- Pizzas italiennes illimitées comprisent dans l'inscription
- Boissons offertes par nos partenaires

Les jeux compétitif

Les jeux seront instalés sur les 80 machines de l'UFR réservées pour la LAN.
Toutefois, nous vous recommandons d'amener vos claviers, souris et tapis.
Vous pouvez néamoins apporter votre propre matériel sous certaines conditions ( voir règlement )
